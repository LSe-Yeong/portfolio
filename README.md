## 🔎 프로젝트 소개  

위 프로젝트는 개인 포트폴리오를 React를 이용하여 구현한 프로젝트 입니다.   
### 포함 내용
1. 화면 구성
2. 사용 기술 스택 
3. 디렉토리 구조 및 패키지 역할
4. 기술적 이슈 및 해결 과정
5. 프로젝트 팀원 및 역할

배포된 포트폴리오 웹 사이트는 다음 링크에서 확인할 수 있습니다.
